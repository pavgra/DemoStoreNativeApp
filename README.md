# DemoStoreNativeApp

## Install
```
npm i
npx react-native link
cd ios
pod install
```

## Run

Run instructions for iOS:
```
cd "DemoStoreNativeApp" && npx react-native run-ios
```
or
- Open DemoStoreNativeApp/ios/DemoStoreNativeApp.xcworkspace in Xcode or run "xed -b ios"
- Hit the Run button
